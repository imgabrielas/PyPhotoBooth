# Python Photobooth

This app opens your camera, counts down from 10 seconds, takes 3 photos, and builds a vertical photobooth strip. Each photo appears on the right-side panel after capture. There is a 5-second pause between photos so you can change pose.

## Install

```bash
python3 -m pip install -r requirements.txt
```

## Controls

- Click `SAVE STRIP` after the third photo to save the final photobooth image.
- Press `S` after the third photo as a keyboard shortcut to save.
- Press `Q` any time to quit.

Saved photobooth images are written to the `outputs` folder.
